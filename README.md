# InstaDemo
A simple demo for Instagram backend  
一个简单的Instagram demo

## How to Run the Demo
clone代码到本地，Windows平台打开Powershell，macOS或Linux打开控制台。切换路径到解压后manage.py所在位置，输入 "python manage.py runserver" 。
